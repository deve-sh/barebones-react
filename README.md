# Barebones React

A zero-config, parcel-based barebones React app.

### Getting Started

```bash
git clone https://github.com/deve-sh/barebones-react.git
cd barebones-react
npm install
npm run dev
```

### Build your app

```bash
npm run build
```

### Serve your app

After you've built your app run:

```bash
npm run serve
```
